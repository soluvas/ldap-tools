h2. Import from vCard (WITHOUT Photo) File(s)

Syntax:

bc. ./ldapcli.sh import-vcard USERS_DN FILE [FILE ...]

Example:

bc. ./ldapcli.sh import-vcard ou=users,dc=berbatik,dc=com output/vcard/facebook_1225303239_santi_mutiara_sari.vcf output/vcard/facebook_1782503068_fitrya_murbarani.vcf

h2. Import from vCard (WITH Photo) File(s)

Syntax:

bc. ./ldapcli.sh import-vcardphoto USERS_DN FILE [FILE ...]

Example:

bc. ./ldapcli.sh import-vcardphoto ou=users,dc=berbatik,dc=com output/vcard/facebook_1225303239_santi_mutiara_sari.vcf output/vcard/facebook_1782503068_fitrya_murbarani.vcf
